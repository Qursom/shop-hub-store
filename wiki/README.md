The main task list. Start here. It contains:

## Getting Started

1. Read [candidate-tasks.md](./candidate-tasks.md) to understand what needs to be built
2. From the repo root, install dependencies with `npm run install:all`, then start the API with `npm run start:api` and the frontend with `npm run start:frontend`. See [frontend/README.md](../frontend/README.md) for more details
3. Refer back to these docs as you implement each feature

## Assignment Overview

- **Framework**: Angular 21 with TypeScript (strict mode)
- **Technologies**: RxJS, TailwindCSS, Reactive Forms
- **Testing**: Vitest + jsdom
- **AI Tool Usage**: Allowed and encouraged

## FAQ

**Q: Can I use AI tools?**\
A: Yes, AI tool usage is allowed and encouraged.

**Q: What if I get stuck?**\
A: Review the JSDoc comments and TODO hints in the service/component stubs, check the Angular documentation, or ask your interviewer.

**Q: How long should this take?**\
A: Aim for 1–2 hours across all features. Manage your time across tasks.

**Q: Can I refactor existing code?**\
A: Yes, if it improves quality.
